## 使用方式

### 查看 Kit 文档

每个 Kit 的文档包含：
- Kit 概述和重要说明
- 导入方式
- API 参考
- 使用示例
- 最佳实践
- 常见问题
- 权限要求
- 参考资料

### 使用 Kit 示例代码

示例代码位于 `/ykit/src/main/ets/kits/<kit-dir>/` 目录，遵循以下结构：

```typescript
/**
 * <Kit Name> 示例页面
 */

import { <exports> } from '@kit.<KitName>'
import { hilog } from '@kit.PerformanceAnalysisKit'

@ObservedV2
export class <Kit>ExampleVM {
  // ViewModel 代码
}

@ComponentV2
export struct <Kit>ExamplePage {
  // 页面代码
}

@Builder
export function <Kit>ExamplePageBuilder() {
  <Kit>ExamplePage()
}
```
